# Identity and Security

This discovery domain explains how the environment is trusted, governed, and secured.

## Scope

It should capture:

- AD and cluster object context
- cluster and Arc identity model
- RBAC tied to the deployment
- certificates and TLS posture
- BitLocker, secured-core, WDAC, Defender, and audit posture
- local admin and policy-related security signals

## Why It Matters

Security posture is part of both technical understanding and customer-ready documentation.
