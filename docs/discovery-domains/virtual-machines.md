# Virtual Machines

This discovery domain explains what the Azure Local platform is hosting.

## Scope

It should capture:

- VM inventory and state
- CPU and memory allocation
- disk and network configuration
- host placement and anti-affinity context
- replication and checkpoint posture
- Arc-managed VM context where applicable
- workload density and overcommit summary

## Why It Matters

Ranger should not stop at platform inventory. It needs to show what the platform is actually running.
