# Performance Baseline

This discovery domain captures short-horizon operational signals that help teams understand the current state of the environment.

## Scope

It should capture:

- compute utilization
- storage throughput, IOPS, latency, and cache signals
- network utilization and errors
- active faults and operational alerts
- recent important event patterns

## Why It Matters

Ranger should provide not only static inventory but also a baseline view of health and operating condition.
