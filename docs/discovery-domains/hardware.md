# Hardware

This discovery domain describes the physical reality of the Azure Local platform.

## Scope

It should capture:

- manufacturer, model, serial, and asset data
- BIOS, firmware, and UEFI posture
- BMC and out-of-band management information
- CPU, memory, DIMM, GPU, disk, and NIC inventory
- TPM and host security hardware state
- vendor classification for OEM integration logic

## Why It Matters

Ranger cannot describe an Azure Local deployment completely unless it can explain what hardware the platform is actually built on.
